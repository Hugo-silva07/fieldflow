# FieldFlow 🚀

Plataforma de gestão operacional e automação de fluxos desenvolvida para distribuição inteligente de demandas, controle operacional e eficiência de atendimento.

---

# Funcionalidades

- Distribuição automática de demandas
- Controle de status operacionais
- Sistema de standby e reatribuição
- Dashboard operacional
- APIs REST
- Controle de autenticação
- Histórico de atividades
- Métricas de produtividade
- Automação de fluxos operacionais

---

# Stack utilizada

## Frontend
- Next.js
- React
- JavaScript
- TailwindCSS

## Backend
- Django REST Framework
- FastAPI
- Python

## Banco de dados
- SQLite

## Outros
- APIs REST
- Autenticação
- Workflows operacionais
- Lógica de filas
- Automação de processos

---

# Objetivo do projeto

O FieldFlow foi desenvolvido com foco em automação operacional, gestão inteligente de demandas e melhoria de produtividade em ambientes corporativos.
